Funções e Definições

* Relação

Conjuntos A,B

Produto cartesiano: A x B { (a,b), a E A; b E b}

A = {1,2,3}
B = {4,6}
A x B { (1,4); (1,6); (2,4); (2,6); (3,4); (3,6) }

* Relação (subconjunto):  |R c AxB| => R {(2,4),(3,6)}  

* Função (relação especial) => R será uma função:  se pra todo elemento a pertenente 
ao conjunto A existir um b, tal que (a,b) pertence a relação
____________
|f:  A -> B|
|    a -> b|  f(a) = b
------------
    |     |
    |     Contradominio
    Dominio

Imagem (Os elementos que levão flechinha)

Tipos de Funções{
- Injetoras (So quando os elementos da imagem receberem uma unica flechinha)
- Sobrejetoras (Possuem todos os elementos do Contradominio associados a pelo menos
um elemento do Dominio)
- Bijetoras(Para uma função ser Bijetora, não é possível dois elementos diferentes
do domínio sejam associados  a um mesmo elemento da imagem)

}
