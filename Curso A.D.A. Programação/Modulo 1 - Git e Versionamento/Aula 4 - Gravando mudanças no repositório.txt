Estudos do GIT:

* Quando a gente ta com nosso arquivo no repositório ele pode estar em alguns estados, sendo eles{
    - Unmodified: Arquivo ja maquepado por GIT, ja passou pelo estado de stage e ja tenha dado commit no arquivo 
    - Modified: Após mudar um arquivo Unmodified ele torna-se Modified
    - Stage: Area preparatória para dar o commit
}

* Digitando 'git status', ele mostra com a branch a gente está
como também mostra se está atualizado. Mostra também, que nada
foi mandado para o 'stage', para ser commit*

Comandos {
    - git add 'Arquivo.formato' (ex. git add .\README.md)
    - git status (mostrará se tem arquivos modificados a serem enviados)
}
