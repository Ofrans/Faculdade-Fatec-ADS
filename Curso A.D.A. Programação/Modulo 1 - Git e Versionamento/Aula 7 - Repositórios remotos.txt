comandos {
    - git add 'nome do arquivo'
    - git commit -m "mensagem" (faz um 'checkpoint')
    - git remote (ve tudo que tem em relação aos remotos)
    - git push origin master (enviará para a master)
    - git pull (puxará tudo que está nosso repositório remoto e juntará seu código com o que estará no seu repositório remoto)
    - git fetch (quais modificações estão para ser feitas)
    }
