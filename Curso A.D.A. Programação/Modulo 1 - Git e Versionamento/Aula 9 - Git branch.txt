Head indica em que Branch estamos

comando{
    - git branch 'nome da nova branch' (Cria uma nova Branch)
    - git log --oneline --decorate (ele nos indica que a gente ta com Head em
    uma determinada branch)
    - git checkout 'nome da branch que você quer ir' (troca a branch que o HEAD 
    estará indicando)
}
* Ao trocar de branch, os commits feito na branch anterior, não afetam os commits
da branch atual.
