Comandos {
    - git diff (mostra o que foi removido e incrementado no codigo)
    - git add (tiramos do estado de modificado para a area de staged)
    - git diff -- staged (Mostra o que foi alterado, removido e incrementado na area de stage)
    - git commit -m "mensagem" (commita o codigo e a mensagem do commit)
}
